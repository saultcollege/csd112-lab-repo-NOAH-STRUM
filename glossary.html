Glossary

Block-level elements
They form a visible block, appear on their own line and usually have other elements nested inside them.
Inline elements
They appear inline with text content and they can't usually contain block-level elements.
Character sets
Allows you to use symbols for almost any language, including emoji.
h#
This is used to mark headings in a document. There are 6 levels of headings.
address element
This element can be used to indicate contact information.
